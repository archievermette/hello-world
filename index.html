Let’s go! On code!

Ouvre ce fichier dans ton fureteur favori. Déjà, ça se fait bien, non? Mais ce n’est pas du « vrai » codage, car HTML est un langage de balise, pas de programmation

Donc essaie d’ajouter un script de type « Hello world! » avec JavaScript. Ce langage est le plus facile de tous à exécuter, car tout ce dont tu as besoin, c’est un fureteur (Python ne s’exécute pas dans un fureteur comme le fait JavaScript). Tu ouvres le fichier HTML contenant un script de type JavaScript dans ton fureteur, et tu peux voir s’exécuter le résultat. C’est tout!
